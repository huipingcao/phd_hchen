DATASETS
The datasets folder contains the 32 benchmark time series. Each file contains one time series. The desc.txt file contains additional information.

EXPERIMENTS
The experiments folder contains the raw results from our experiments that the jupyter notebooks analyze.

NOTEBOOKS
The notebooks folder contains jupyter notebooks for our experiments and analyze the results.

SRC
The src folder contains the ClaSP code in (clasp.py) and a test that runs ClaSP on the benchmark time series (clasp_test.py). You can simply run "python clasp_test.py" and review the results.
